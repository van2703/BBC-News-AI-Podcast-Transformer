# BBC News AI Podcast Transformer

## Project Overview

**BBC News AI Podcast Transformer** is a project that automatically converts news articles from BBC into AI-generated podcast episodes.

The system collects news from BBC, summarizes the content using Artificial Intelligence, generates a podcast script, converts the script into audio, and uploads the podcast to a website where users can listen to it.

## Objectives

* Automatically collect news articles from BBC
* Use AI to summarize the content
* Generate a podcast-style script
* Convert the script into audio using Text-to-Speech
* Upload the podcast to a website for users to listen

## System Workflow

The system operates in the following steps:

1. **News Collection** – Fetch news articles from BBC News
2. **AI Summarization** – AI summarizes the article
3. **Script Generation** – Create a podcast script from the summary
4. **Audio Generation** – Convert the script into audio using Text-to-Speech
5. **Publishing** – Upload the podcast audio to the website

## Technologies Used

* **Programming:** Python
* **AI Processing:** Gemini API
* **Web Scraping:** BeautifulSoup
* **Audio Generation:** Text-to-Speech (TTS)
* **Web Platform:** HTML, CSS, WordPress
